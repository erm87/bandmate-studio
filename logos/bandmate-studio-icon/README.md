# BandMate Studio app icon

## Source of truth
`concept-B-wave.svg` — the vector master. Edit this if you want to change the design.

## How to regenerate the production icon set

Use Tauri's own icon CLI from a single high-resolution **master PNG**, NOT raw ImageMagick. It handles all the platform-specific outputs (`.icns`, `.ico`, all PNG sizes, iOS, Android) and gets the bit depth right.

```bash
# 1. Render a single master PNG from the SVG (1024 is the standard).
cd /path/to/bm-loader-rebuild
convert -background none -size 1024x1024 -depth 8 \
  logos/bandmate-studio-icon/concept-B-wave.svg \
  logos/bandmate-studio-icon/master-1024.png

# 2. Hand the master to Tauri's icon tool. Overwrites src-tauri/icons/ in place.
pnpm tauri icon logos/bandmate-studio-icon/master-1024.png

# 3. Rebuild.
pnpm tauri build
```

The `-depth 8` flag is critical when using ImageMagick — its Q16 build defaults to 16-bit-per-channel output, which Tauri's icon decoder cannot read. See the lesson learned below.

## Lesson learned (2026-05-12)

The first version of this icon set was generated by manually rasterising the SVG to every Tauri size via `convert` without `-depth 8`. The resulting PNGs were valid (Finder and Preview opened them fine; macOS `iconutil` produced an `.icns` without complaint) but were 16-bit/channel RGBA. Tauri's icon decoder hard-assumes 8-bit and panicked at app launch with:

```
Failed to setup app: runtime error: invalid icon:
The specified dimensions don't match the buffer length
```

…during the `NSApplicationDidFinishLaunching` observer, which presents as a SIGABRT in the crash report (release builds have `panic = "abort"` in Cargo.toml, so the panic becomes an immediate process abort).

Why this was hard to spot earlier:

- Dev builds (`pnpm tauri dev`) **do not bundle** `tauri.conf.json`'s `bundle.icon` set, so the bug never surfaced during ordinary dogfood testing.
- `iconutil` and Preview happily accept 16-bit PNGs.
- The crash report's symbols were stripped (release build), so the stack didn't say "icon".

The diagnostic path was: run the bundled binary directly from terminal to capture stderr → see the Rust panic message → look at PNG bit depth.

## Files in this directory

- `concept-B-wave.svg` — the shipping vector master.
- `concept-A-knob.svg`, `concept-C-ripples.svg` — earlier concept sketches, kept for reference.
- `icon.iconset/` — Apple's iconset format directory (10 PNGs at the exact filenames `iconutil` requires). Generated as an intermediate when bypassing `pnpm tauri icon`. Not needed if you use the Tauri CLI flow above.
- `preview-*.png`, `polished-*.png`, `tiny-*.png`, `comparison-*.png` — render previews from the design phase. Not used at build time.
